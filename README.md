## Overview

This project aims to compare different cache eviction policies, FIFO (First in First Out), LRU (Least Recently Used) and OPTFF (Belady's Farthest-in-Future, optimal offline). 

---

## How to Run It

Make sure you are in the "src" folder and run: 

"python main.py <input_file>"

The program will take the data from the input file in the "data" folder and print 

FIFO  : <number_of_misses>
LRU   : <number_of_misses>
OPTFF : <number_of_misses>
